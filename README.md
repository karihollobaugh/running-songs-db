# Running Songs Database API

A RESTful API built with Java and Spring Boot for runners who want to curate playlists organized by BPM, genre, and artist.

## Tech Stack
- Java / Spring Boot
- PostgreSQL
- Tested with Postman

## Features
- GET all songs in the database
- GET a single song by ID
- POST one or multiple songs at once
- DELETE a song by ID

## Architecture
Built using a four-layer Spring Boot structure: Controller → Service → Repository → Data Model
